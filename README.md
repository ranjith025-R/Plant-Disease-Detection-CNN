# 🌿 Plant Disease Detection System

## 📌 Project Overview
This project uses **Deep Learning** to identify 38 different types of plant diseases from images of leaves. The goal is to provide an automated tool for farmers to detect crop issues early.

## 🚀 Technical Highlights
- **Architecture:** Leveraged **MobileNetV2** via Transfer Learning for high efficiency.
- **Preprocessing:** Used `ImageDataGenerator` for rescaling and data augmentation.
- **Framework:** Built with **TensorFlow** and **Keras** in a Jupyter Notebook environment.

## 🛠️ Tools Used
- Python
- TensorFlow / Keras
- NumPy & Pandas
- Matplotlib (for performance visualization)
